## Функциональность

### Математические функции
- **Факториал** (итеративный и рекурсивный)
- **Числа Фибоначчи** (итеративный и рекурсивный)

### Алгоритмы сортировки
- **Пузырьковая сортировка** (Bubble Sort)
- **Быстрая сортировка** (Quick Sort) 
- **Сортировка подсчетом** (Counting Sort)
- **Поразрядная сортировка** (Radix Sort)
- **Блочная сортировка** (Bucket Sort)
- **Пирамидальная сортировка** (Heap Sort)

**Важно:** Все алгоритмы сортировки работают только с целыми числами. Входные данные преобразуются в целые числа перед сортировкой.

### Структуры данных
- **Стек** на связном списке с поддержкой операций:
  - `push` - добавление элемента
  - `pop` - удаление и возврат верхнего элемента
  - `peek` - просмотр верхнего элемента
  - `is_empty` - проверка на пустоту
  - `__len__` - получение размера

## Установка и запуск

### Требования
- Python 3.8+
- Зависимости из `requirements.txt`

```bash
pip install -r requirements.txt


### Запуск программы

```bash
python src/main.py
```

### Запуск тестов

```bash
# Из корневой директории проекта
pytest tests/ -v

# Или с указанием конкретного файла
pytest tests/test_expression.py -v

# Альтернативный вариант через python -m
python -m pytest tests/ -v
```

## Использование

### Главное меню

После запуска программы доступны следующие команды:

```
ГЛАВНОЕ МЕНЮ
1 - Факториал (итеративный)
2 - Факториал (рекурсивный)
3 - Число Фибоначчи (итеративный)
4 - Число Фибоначчи (рекурсивный)
5 - Сортировка пузырьком
6 - Быстрая сортировка
7 - Сортировка подсчетом
8 - Поразрядная сортировка
9 - Блочная сортировка
10 - Пирамидальная сортировка
11 - Работа со стеком
exit - Выход из программы
```

### Примеры использования

**Факториал:**
```
Введите команду: 1
Введите число: 5
120
```

**Сортировка:**
```
Введите команду: 5
Введите числа через пробел: 3 1 4 2 5
1 2 3 4 5
```

**Работа со стеком:**
```

МЕНЮ СТЕКА
push <значение> - Добавить элемент в стек

peek - Показать верхний элемент без удаления

Введите команду для стэка: peek
Верхний элемент: 10
